## Generating Screenshots

Screenshots can be captured using:
1. iOS Simulator with `xcrun simctl io screenshot`
2. Android Emulator with `adb exec-out screencap -p > screenshot.png`
3. Physical devices
4. Third-party tools like Fastlane Snapshot or Screenshots
